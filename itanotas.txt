Notas Manual Comandos 

ls.- muestra una lista del contenido del directorio donde se encuentra. Ej. ls -al

echo.-

cd.- permite moverse de un directorio a otro. Ej. cd Archivos

cd .. .- nos regresa a un diectorio inmediatamente arriba del que nos encontramos. Ej. cd ..

pwd.- muestra el nombre del directorio en que se encuentra trabajando. Ej. pwd

file.- permite obtener informasion sobre el tipo de archivo. Ej. file music

man.- nos dirije al manual sore cada comando y permite conocer la funcion de cada uno. Ej. man pwd man mkdir

apropos / man -k .- busca algun comando o archivo. Ej. apropos txt

mkdir.- crea un nuevo directorio dentro del que nos encontremos. Ej. mkdir Archivos

rmdir.- borra un directorio. Ej. rmdir Archivos

touch.- permite crear un nuevo archivo. Ej. touch archivo.txt

cp.- copiar archivos o directorios. Ej. cp archivo.txt Archivos

mv.- mover archivos o directorios. Ej. mv archivo.txt Documents

rm.- borrar archivos o directorios. Ej. rm archivo.txt

cat.- encadenar e imprimir archivos.
